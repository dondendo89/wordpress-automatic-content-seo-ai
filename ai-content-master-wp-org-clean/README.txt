Professional AI-powered content generation plugin with freemium model. Generate SEO-optimized content, meta descriptions, tags, and excerpts using Google Gemini AI.

== Description ==

AI Content Master is a powerful WordPress plugin that leverages Google Gemini AI to automatically generate high-quality, SEO-optimized content for your website. Perfect for bloggers, content creators, and website owners who want to streamline their content creation process.

= Key Features =

* **AI Content Generation**: Generate high-quality, SEO-optimized content using Google Gemini API
* **Freemium Model**: 10 free generations per month, then upgrade to premium service
* **Meta Descriptions**: Create compelling meta descriptions for better SEO
* **Tag Generation**: Generate relevant tags automatically
* **Excerpt Creation**: Create engaging excerpts for posts
* **Bulk Processing**: Process multiple posts at once with background jobs
* **Gutenberg Integration**: Native Gutenberg editor support
* **Multi-language Support**: Generate content in multiple languages (EN, IT, ES, FR, DE)
* **WooCommerce Compatible**: Works with WooCommerce products
* **Professional UI**: Clean, modern admin interface

= How It Works =

1. **Free Tier**: Get 10 AI content generations per month at no cost
2. **Easy Setup**: Simply add your Google Gemini API key in the plugin settings
3. **Generate Content**: Use the meta box on any post to generate content, meta descriptions, tags, or excerpts
4. **Bulk Processing**: Process multiple posts at once with our bulk generation feature
5. **Upgrade**: When you need more generations, upgrade to our premium service

* WordPress 6.0 or higher

= Getting Started =

1. Install and activate the plugin
2. Go to Settings > AI Content Master
3. Add your Google Gemini API key
4. Configure your preferred settings
5. Start generating content!

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ai-content-master/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > AI Content Master to configure your API key
4. Start generating content!

== Frequently Asked Questions ==

= How many free generations do I get? =

You get 10 free AI content generations per month. This resets automatically on the 1st of each month.

= Do I need a Google Gemini API key? =

Yes, you need a free Google Gemini API key to use the plugin. You can get one from Google AI Studio.

= What languages are supported? =

The plugin supports content generation in English, Italian, Spanish, French, and German.

= Does it work with WooCommerce? =

Yes! The plugin is fully compatible with WooCommerce products and will generate appropriate tags for products.

= Can I process multiple posts at once? =

Yes, the plugin includes a bulk processing feature that allows you to process multiple posts simultaneously.

= Is there a premium version? =

The plugin includes a freemium model. After using your 10 free generations, you can upgrade to our premium service for unlimited generations.

= How do I upgrade to premium? =

When you run out of free generations, you'll see an upgrade option that redirects to our external premium service.
