WP Gemini Content Generator

Overview

WP Gemini Content Generator is a powerful WordPress plugin that automatically generates SEO-friendly, long-form content descriptions for your posts, pages, and WooCommerce products using Google's Gemini AI.

Features

  • Generate 2000+ character descriptions with one click
  • Support for Posts, Pages, and WooCommerce Products
  • 12 language support (English, Italian, Spanish, French, German, Portuguese, Russian, Japanese, Korean, Chinese, Arabic, Hindi)
  • HTML-formatted content with SEO structure
  • Bulk generation for multiple posts/pages
  • Replace or append content modes
  • Optional emoji and icons support
  • Product-specific descriptions for WooCommerce
  • Secure API key storage

Installation

  1. Upload the plugin files to your WordPress site
  2. Activate the plugin through the 'Plugins' menu
  3. Go to Settings → Gemini Generator
  4. Enter your Google Gemini API key
  5. Configure your preferences

Getting Your API Key

To use this plugin, you need a Google Gemini API key:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the generated key
  5. Paste it in the plugin settings

Configuration

API Settings

  • Gemini API Key: Your Google Gemini API key
  • Content Language: Select the language for generated content
  • Enable on Post Types: Choose which post types to enable the plugin on
  • Content Mode: Replace existing content or append to it
  • Emoji & Icons: Enable/disable emoji and icons in generated content

Usage

Single Post/Page Generation

  1. Edit any post, page, or product
  2. Look for the "Gemini Content Generator" meta box
  3. Click "Generate Long Description" (or "Generate Product Description" for products)
  4. Wait for the content to be generated and inserted

Bulk Generation

  1. Go to Settings → Gemini Generator
  2. Set your batch size (1-20 posts per request)
  3. Click "Run Bulk Generation"
  4. Monitor the progress in the status area

Requirements

  • WordPress 6.0 or higher
  • PHP 7.4 or higher
  • Google Gemini API key
  • Internet connection for API calls

Security

This plugin follows WordPress security best practices:

  • All user inputs are sanitized and validated
  • AJAX requests are protected with nonces
  • Capability checks ensure only authorized users can use the plugin
  • API keys are stored securely in WordPress options
  • HTML content is sanitized with wp_kses()
