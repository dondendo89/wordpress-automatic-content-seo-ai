üîß Installation Guide

Method 1: WordPress Admin (Recommended)

Step 1: Download Plugin

Download the plugin ZIP file from CodeCanyon or your purchase confirmation email.

Step 2: Upload Plugin

Go to Plugins ‚Üí Add New ‚Üí Upload Plugin in your WordPress admin.

Step 3: Install & Activate

Upload the ZIP file, click Install Now, then Activate Plugin.

Method 2: FTP Upload

Step 1: Extract Files

Extract the ZIP file to your computer to get the wp-gemini-content-generator folder.

Step 2: Upload via FTP

Upload the folder to /wp-content/plugins/ on your server.

Step 3: Activate

Go to Plugins in WordPress admin and activate the plugin.

‚úÖ Installation Complete! The plugin is now installed and ready for configuration.

üìù Usage Instructions

Single Content Generation

Step 1: Edit Post/Page

Go to any post, page, or product editor in WordPress.

Step 2: Generate Content

Look for the "Gemini Content Generator" meta box and click "Generate Content".

Step 3: Review & Save

Review the generated content in the preview and save your post.

Bulk Content Generation

Step 1: Go to Settings

Navigate to Settings ‚Üí Gemini Generator ‚Üí Bulk Generation tab.

Step 2: Configure Options

Select post types, set batch size, and choose processing options.

Step 3: Start Processing

Click "Start Bulk Generation" and monitor progress.

Meta Description Generation

Individual Generation

Use the "Generate Meta Description" button in the post editor.

Bulk Generation

Enable "Include meta descriptions and tags" in bulk processing.

Tag Generation

Individual Generation

Use the "Generate Tags" button in the post editor.

Bulk Generation

Enable "Include meta descriptions and tags" in bulk processing.

üîß Troubleshooting

Common Issues

Plugin Not Working After Activation

Possible Causes:
  • Missing cURL or JSON extensions
  • Invalid Gemini API key
  • PHP errors in error logs
  • Insufficient server resources
# Check PHP extensions php -m | grep -E "(curl|json)" # Check WordPress error logs tail -f /path/to/wordpress/wp-content/debug.log

Content Not Generating

Possible Causes:
  • Invalid or expired API key
  • Insufficient API credits
  • Network connectivity issues
  • Server firewall blocking requests
# Test API connectivity curl -X POST "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash-lite:generateContent?key=YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"contents":[{"parts":[{"text":"Test"}]}]}'

Bulk Processing Fails

Possible Causes:
  • Server timeout limits
  • Memory exhaustion
  • Database connection issues
  • Large batch sizes
# Reduce batch size in settings # Check server timeout settings # Monitor server resources during processing

Error Messages

"Missing Gemini API key"

Solution: Set your API key in Settings ‚Üí Gemini Generator

"No posts found to process"

Solution: Check post type selection and existing content

"API Error"

Solution: Verify your Gemini API key and credits

"Network error"

Solution: Check internet connection and server configuration

Performance Optimization

For Large Sites:
  • Reduce batch size to 3-5 items
  • Process during off-peak hours
  • Monitor server resources
  • Consider upgrading hosting plan
